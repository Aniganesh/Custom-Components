Timed counter

Introduction

The timed-counter element is used to create a customisable linear counter. It has no mandatory attributes. It has the following optional attributes:

  • interval
  • min
  • max
  • step
  • loop
  • onend

interval

This is used to set the time after which the value displayed by the counter is modified. The time is specified in seconds. The default value is 1. i.e., the value will be altered after 1 second.

min

This attribute specifies the minimum value displayed by the counter. It is also the value the counter displays at the start. The default value is 0. i.e., the counter displays 0 on start. It expects a numeric value. If a non-number value is provided, it falls back to the default value.

max

This attribute specifies the maximum value displayed by the counter. It is also the value the counter displays at the end. The default value is 100. This too expects a numeric value. If a non-numeric value is provided, it falls back to the default value.

step

This attribute specifies the increment to the value displayed by the counter. Like the other attributes it expects a numeric value. The default value is 1. Like the other attributes, it falls back to the default value if provided a non-numeric value.

loop

The presence of the attribute makes the counter repeat counting from the minimum value once the maximum value is reached. It expects no values.

onend

This attribute specifies a function to be called at the end of the timer. It expects a non-anonymous function. A non function value will throw a TypeError. If the function does not exist, it will throw a reference error.
